================================================================================
  BERANS AUTO — как запустить проект (фронтенд + бэкенд)
================================================================================

Нужно установлено:
  • Python 3 (рекомендуется 3.10+)
  • Node.js и npm

Структура проекта:
  BernensAuto/
    bernansauto/
      backend/     — Django-бэкенд (API, админка)
      frontend/    — React-фронтенд (интерфейс)

--------------------------------------------------------------------------------
  1. БЭКЕНД (Django)
--------------------------------------------------------------------------------

1.1. Создать и активировать виртуальное окружение Python

    В корне проекта (BernensAuto):

    Windows (PowerShell или cmd):
      python -m venv .venv
      .venv\Scripts\activate

    После активации в начале строки появится (.venv).

1.2. Установить зависимости

    Перейти в папку бэкенда и установить пакеты:

      cd bernansauto\backend

    Установить:
      pip install -r requirements.txt

1.3. Настроить RAdmin

    • скачать и запустить программу
    • подключиться к BernansAuto
    Сеть - Подключиться к сети - Название сети: BernansAuto, Пароль: BernansAuto

1.4. Миграции и запуск сервера

    В папке bernansauto/backend выполнить:

      python manage.py runserver

    Бэкенд будет доступен по адресу:  http://127.0.0.1:8000/
    Админка Django:                   http://127.0.0.1:8000/admin/
    API (Swagger):                    http://127.0.0.1:8000/swagger/

--------------------------------------------------------------------------------
  2. ФРОНТЕНД (React)
--------------------------------------------------------------------------------

2.1. Установить зависимости

    Открыть второй терминал. Перейти в папку фронтенда:

      cd bernansauto\frontend

    Выполнить:

      npm install

2.2. Запустить фронтенд

      npm start

    Откроется браузер с приложением по адресу:  http://localhost:3000

    Фронтенд по умолчанию обращается к бэкенду на порту 8000 (CORS уже настроен
    для http://localhost:3000 в settings.py бэкенда).

--------------------------------------------------------------------------------
  3. ИТОГО: как запускать каждый раз
--------------------------------------------------------------------------------

  Терминал 1 (бэкенд):
    cd bernansauto\backend
    .venv\Scripts\activate    (если ещё не активировано)
    python manage.py runserver

  Терминал 2 (фронтенд):
    cd bernansauto\frontend
    npm start

  В браузере открыть:  http://localhost:3000  — главная страница с каталогом
  и формой входа/регистрации.

--------------------------------------------------------------------------------
  Возможные проблемы
--------------------------------------------------------------------------------

  • "ModuleNotFoundError: No module named 'django'" — активируйте venv и
    установите зависимости (шаг 1.2).

  • Ошибка подключения к PostgreSQL — проверьте, что сервер PostgreSQL запущен,
    база BernansAuto_DATABASE создана, логин/пароль в settings.py совпадают.

  • Фронтенд не видит API — убедитесь, что бэкенд запущен на http://127.0.0.1:8000
    и в backend/bernansauto/settings.py в CORS_ALLOWED_ORIGINS есть "http://localhost:3000".
